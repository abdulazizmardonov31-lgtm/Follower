# 🌞 Solar Follower (Sun Tracking System)

## 📌 Overview
Solar Follower is a project developed during the English for IT Camp.  
It is designed to maximize solar energy efficiency by automatically adjusting the position of a solar panel to follow the sun throughout the day.

## 🚀 Features
- Automatic sun tracking using light sensors (LDR)
- Real-time adjustment of panel angle
- Increased energy efficiency compared to fixed panels
- Simple and low-cost implementation

## ⚙️ How It Works
The system uses two light-dependent resistors (LDRs) to detect sunlight intensity.  
A microcontroller (Arduino) compares the values and rotates the panel toward the brighter side using a servo motor.

## 🛠️ Technologies Used
- Arduino (C/C++)
- LDR Sensors
- Servo Motor
- Basic Electronics
